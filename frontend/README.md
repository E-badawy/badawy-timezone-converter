# Badawy Timezone Converter - Frontend

React-based user interface for the timezone conversion application.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm start
```

The app will open at `http://localhost:3000`

## Features

- **Converter**: Convert time between any two timezones
- **Current Time**: Check current time in any timezone with live updates
- **Comparator**: Compare current time across multiple timezones simultaneously

## Styling

Modern gradient-based UI with:
- Responsive design for mobile and desktop
- Purple theme (#667eea to #764ba2)
- Smooth transitions and hover effects
- Clean, intuitive interface
